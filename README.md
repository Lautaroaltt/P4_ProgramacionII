# Sistema de Gestión de Estudiantes

Proyecto hecho con Flask para gestionar estudiantes de forma simple. Permite listar, agregar, editar, buscar y eliminar registros.

## Tecnologias usadas

- Python
- Flask
- HTML
- CSS
- Git

## Como ejecutar

1. Crear y activar el entorno virtual:

```bash
python -m venv venv
source venv/bin/activate
```

En Windows:

```bash
venv\Scripts\activate
```

2. Instalar dependencias:

```bash
pip install -r requirements.txt
```

3. Configurar la base de datos PostgreSQL con las variables de entorno del proyecto.

4. Ejecutar la app:

```bash
python app.py
```

5. Abrir en el navegador:

```text
http://127.0.0.1:5000
```
